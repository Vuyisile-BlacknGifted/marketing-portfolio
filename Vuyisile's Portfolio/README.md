# Vuyisile Phika Portfolio

A clean, modern growth marketing portfolio built with Next.js and Tailwind CSS.

## Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Framer Motion (subtle section animations)

## Run locally

1. Install Node.js 18.18+ (or 20+).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000).

## Content updates

Main portfolio content lives in:

- `src/app/page.tsx`

Reusable UI components live in:

- `src/components/cards.tsx`
- `src/components/SectionHeading.tsx`
- `src/components/AnimatedSection.tsx`
